## 서론

&nbsp;&nbsp; 이전 글에서는 자연어 처리(NLP)에서 중요한 문제인 할루시네이션(hallucination) 현상과 이를 완화하기 위한 접근 방식을 소개했습니다. 또한, XLM 모델을 사용해 사용자 질문에 적절한 응답을 반환하는 파이프라인을 구축했습니다.

이번 글에서는 해당 파이프라인을 Django와 Docker를 활용해 웹 애플리케이션으로 배포하는 방법을 설명하겠습니다. 배포 과정은 다음과 같은 순서로 진행됩니다.

1. 모델과 Django 애플리케이션 통합
2. 서비스 성능 개선
3. Docker 컨테이너화 및 배포

## Dockerfile 작성
```bash
FROM kuper0201/pytorch:1.13.1
RUN pip install --upgrade pip
RUN pip install pandas django sentence-transformers==2.2.2
COPY ChatBotDjango /ChatBotDjango
EXPOSE 80
VOLUME ["/ChatBotDjango/data"]
WORKDIR /ChatBotDjango
ENTRYPOINT ["python3", "manage.py", "runserver", "0.0.0.0:80", "--noreload"]
```
